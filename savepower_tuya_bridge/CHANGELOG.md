# Changelog

## [1.0.0] - 2026-01-07

### Added
- Initial release
- Bridge Tuya thermostats to SavePower via WT50 MQTT protocol
- Support for multiple thermostat models (with/without valve sensor)
- Configurable MQTT connection with authentication
- Configurable publish and telemetry intervals
- Device mapping for complex entity naming
- Automatic feature detection per device
- Command acknowledgements per WT50 protocol
- Support for: temperature, mode, thermostat_mode, lock, anti_frost, max_temp, min_temp, calibration, sensor, sound
