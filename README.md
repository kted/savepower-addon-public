# SavePower Tuya Thermostat Bridge

Home Assistant add-on for bridging Tuya thermostats to SavePower platform via MQTT.

## Installation

1. Add this repository to Home Assistant:
   - Settings → Add-ons → Add-on Store → ⋮ → Repositories
   - Add: `https://github.com/YOUR_USERNAME/savepower-addon-public`

2. Install the add-on from the store

3. Configure with your MQTT credentials
